Postgresql client Connection options

PostgreSQL 客户端常用的连接参数
注意:密码是不可能写在命令行上的,设置密码用.pgpass,环境变量

使用环境变量 连接PostgreSQL

  • PGHOST
  • PGPORT
  • PGUSER
  • PGPASSWORD
  • PGDATABASE

使用.pgpass 连接PostgreSQL
在你的HOME目录下创建.pgpass文件,字段可用*
格式: hostname:port:database:username:password
