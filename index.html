Linux NETSTAT SS

netstat ss

NETSTAT

  • -l, вҖ“listening display listening server sockets
  • -a, вҖ“all display all sockets (default: connected)
  • -r, вҖ“route display routing table
  • вҖ“i, вҖ“interfaces display interface table
  • -g, вҖ“groups display multicast group memberships
  • -s, вҖ“statistics display networking statistics (like SNMP)
  • -M, вҖ“masquerade display masqueraded connections
  • -v, вҖ“verbose be verbose
  • -W, вҖ“wide donвҖҷt truncate IP addresses
  • -n, вҖ“numeric donвҖҷt resolve names
  • -e, вҖ“extend display other/more information
  • -p, вҖ“programs display PID/Program name for sockets
  • -o, вҖ“timers display timers
  • -F, вҖ“fib display Forwarding Information Base (default)
  • -C, вҖ“cache display routing cache instead of FIB

зӣ‘еҗ¬22з«ҜеҸЈзҡ„иҝӣзЁӢ

  1. еҲ—еҮәжүҖжңүTCPз«ҜеҸЈ
  2. еҲ—еҮәжүҖжңүзӣ‘еҗ¬зҡ„UDPз«ҜеҸЈ

SS

  • вҖ“n, вҖ“numeric donвҖҷt resolve service names
  • -r, вҖ“resolve : resolve host hostnames.
  • -l, вҖ“listening display listening sockets
  • -o, вҖ“options show timer information
  • -e, вҖ“extended show detailed socket information
  • -m, вҖ“memory show socket memory usage
  • -p, вҖ“processes show process using socket
  • вҖ“s, вҖ“summary show socket usage summary
  • -N, вҖ“net switch to the specified network namespace name
  • -4, вҖ“ipv4 display only IP version 4 sockets
  • -6, вҖ“ipv6 display only IP version 6 sockets
  • вҖ“0, вҖ“packet display PACKET sockets
  • -t, вҖ“tcp display only TCP sockets
  • -S, вҖ“sctp display only SCTP sockets
  • -u, вҖ“udp display only UDP sockets
  • -w, вҖ“raw display only RAW sockets
  • -x, вҖ“unix display only Unix domain sockets
  • -f, вҖ“family=FAMILY display sockets of type FAMILY
  1. еҲ—еҮәзӣ‘еҗ¬зҡ„TCPеҚҸи®®е№¶еҢ…жӢ¬еҜ№еә”зҡ„иҝӣзЁӢ
  2. еҲ—еҮәиҝһжҺҘеҲ°192.168.1.10:443зҡ„ж Ҳ
  3. еҲ—еҮәжүҖжңүsshзӣёе…ізҡ„иҝһжҺҘ

NFS жқғйҷҗ

иҪ¬иҪҪ

й»ҳи®Өи§„еҲҷ

й»ҳи®Өзҡ„жқғйҷҗи§„еҲҷжҳҜпјҢrootз”ЁжҲ·иў«жҳ е°„жҲҗnfsnobodyз”ЁжҲ·пјҢеҜ№дәҺе®ўжҲ·з«ҜжңәеҷЁдёҠе’ҢNFSжңҚеҠЎеҷЁдёҠUIDзӣёеҗҢзҡ„з”ЁжҲ·дјҡеҜ№еә”жҳ е°„пјҢе…¶е®ғйқһrootз”ЁжҲ·иў«жҳ е°„жҲҗnobodyз”ЁжҲ·гҖӮеҪ“rootз”ЁжҲ·и®ҝй—®е…ұдә«зӣ®еҪ•ж—¶жҳҜд»Ҙnfsnobodyз”ЁжҲ·и®ҝй—®е…ұдә«зӣ®еҪ•зҡ„пјҢе…·жңүд»Җд№ҲжқғйҷҗзңӢдёӢе…ұдә«зӣ®еҪ•жқғйҷҗдҫҝзҹҘгҖӮе®ўжҲ·з«ҜжңәеҷЁдёҠе’ҢNFSжңҚеҠЎеҷЁдёҠзҡ„зӣёеҗҢUIDз”ЁжҲ·пјҢд»ҘNFSжңҚеҠЎеҷЁдёҠзҡ„з”ЁжҲ·и®ҝй—®е…ұдә«зӣ®еҪ•пјҢзңӢжҲҸзӣ®еҪ•жқғйҷҗдҫҝзҹҘгҖӮе…¶е®ғйқһrootз”ЁжҲ·еҲҷжҳ е°„жҲҗnobodyз”ЁжҲ·пјҢжңүе•ҘжқғйҷҗдёҖзңӢдҫҝзҹҘгҖӮ

йҖүйЎ№

  • roпјҡй»ҳи®ӨйҖүйЎ№пјҢд»ҘеҸӘиҜ»зҡ„ж–№ејҸе…ұдә«гҖӮ
  • rwпјҡд»ҘиҜ»еҶҷзҡ„ж–№ејҸе…ұдә«гҖӮ
  • no_root_squashпјҡе°Ҷе®ўжҲ·з«ҜдҪҝз”Ёзҡ„жҳҜrootз”ЁжҲ·ж—¶пјҢеҲҷжҳ е°„еҲ°FNSжңҚеҠЎеҷЁзҡ„з”ЁжҲ·дҫқз„¶дёәrootз”ЁжҲ·гҖӮ
  • all_squashпјҡй»ҳи®ӨйҖүйЎ№пјҢе°ҶжүҖжңүи®ҝй—®NFSжңҚеҠЎеҷЁзҡ„е®ўжҲ·з«Ҝзҡ„з”ЁжҲ·йғҪжҳ е°„дёәеҢҝеҗҚз”ЁжҲ·пјҢдёҚз®Ўе®ўжҲ·з«ҜдҪҝз”Ёзҡ„жҳҜд»Җд№Ҳз”ЁжҲ·гҖӮ
  • anonuidпјҡи®ҫзҪ®жҳ е°„еҲ°жң¬ең°зҡ„еҢҝеҗҚз”ЁжҲ·зҡ„UID
  • anongidпјҡи®ҫзҪ®жҳ е°„еҲ°жң¬ең°зҡ„еҢҝеҗҚз”ЁжҲ·зҡ„GID
  • syncпјҡй»ҳи®ӨйҖүйЎ№пјҢдҝқжҢҒж•°жҚ®еҗҢжӯҘпјҢж•°жҚ®еҗҢжӯҘеҶҷе…ҘеҲ°еҶ…еӯҳе’ҢзЎ¬зӣҳгҖӮ
  • asyncпјҡејӮжӯҘпјҢе…Ҳе°Ҷж•°жҚ®еҶҷе…ҘеҲ°еҶ…еӯҳпјҢеңЁе°Ҷж•°жҚ®еҶҷе…ҘеҲ°зЎ¬зӣҳгҖӮ
  • secureпјҡNFSе®ўжҲ·з«Ҝеҝ…йЎ»дҪҝз”ЁNFSдҝқз•ҷз«ҜеҸЈпјҲйҖҡеёёжҳҜ1024д»ҘдёӢзҡ„з«ҜеҸЈпјүпјҢй»ҳи®ӨйҖүйЎ№гҖӮ
  • insecureпјҡе…Ғи®ёNFSе®ўжҲ·з«ҜдёҚдҪҝз”ЁNFSдҝқз•ҷз«ҜеҸЈпјҲйҖҡеёёжҳҜ1024д»ҘдёҠзҡ„з«ҜеҸЈпјүгҖӮ
