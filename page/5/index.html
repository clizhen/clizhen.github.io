Set TimeZone & NTP In Ubuntu

TimeZone

  1. timedatectl list-timezones
  2. sudo timedatectl set-timezone Asia/Shanghai
  3. timedatectl

NTP

  1. sudo apt-get install ntp
  2. sntp –version
  3. sudo vim /etc/ntp.conf 选择NTP server
  4. sudo service ntp restart
  5. sudo service ntp status

SQL XACT_ABORT

When SET XACT_ABORT is ON, if a Transact-SQL statement raises a run-time error, the entire transaction is terminated and rolled back. When SET XACT_ABORT is OFF, in some cases only the Transact-SQL statement that raised the error is rolled back and the transaction continues processing.

Ubuntu 语言设置

  • 查看当前系统语言环境: locale / locale -a
  • 编辑配置文件 sudo vim /etc/default/locale (中文:zh_CN.UTF-8 ,英文:en_US:UTF-8).
  • 安装编码: sudo locale-gen zh_CN.UTF-8.
  • 重启.
